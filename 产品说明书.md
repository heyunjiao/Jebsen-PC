## 二、项目概述

### 2.1 产品定位

| 项目                    | 说明                                                                                                                                                                                     |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **PC 端（JS-demo）**    | C360 客户全景平台，面向 B 端运营、数据、审批、审计及系统管理员，提供客户列表、商机、分群、标签、异常中心、审批、审计、数据看板、催收、系统与监控等能力。                                 |
| **移动端（Jebsen-H5）** | C360 客户画像 H5，支持在企业微信侧边栏及内部系统内嵌使用，面向门店顾问或客户，以单页形式展示与维护客户画像（基础信息、标签、电话、维保/保险/金融、车辆、资产、预约、商机、操作日志等）。 |

### 2.2 技术栈与约束（与合同对照用）

- **PC**：Vue 3.4+、TypeScript、Vite 5+、Element Plus、Pinia、Vue I18n（简中/繁中/英文）。
- **H5**：Vue 3、TypeScript、Vite、Vant 4、Pinia、Vue Router。
- **业务约束**：i18n 先行（禁止中文硬编码）；列表页使用 ProTable/SearchForm；敏感信息脱敏（手机号/证件号）；RBAC 与 `v-auth` 指令；页面响应 <20s、数据处理 T+1（SLA 参考）。

## 五、移动端（Jebsen-H5）功能说明

| 项目              | 说明                                                                                                                                                 |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| **运行环境**      | 企业微信侧边栏、内部系统内嵌（URL 带 token）。                                                                                                       |

详见 **TEST_CASES.md** 中的演示 URL 与预期。
