## 1. 产品概述

### 1.1 模块定位

账号管理模块是 C360 客户全景平台的系统账号管理核心模块，提供系统账号的创建、编辑、删除、权限分配等功能。支持账号状态管理、密码重置、角色分配等操作。

### 1.2 业务价值

- **账号统一管理**：集中管理所有系统账号
- **权限精确控制**：通过角色分配实现权限控制
- **账号安全**：支持账号状态管理和密码重置
- **组织架构关联**：与部门管理关联，支持按部门管理账号

### 1.3 目标客户

- **系统管理员**：创建和管理客户账号
- **部门管理员**：管理本部门账号

## 4. 业务规则

### 4.1 账号创建规则

#### 4.1.1 客户名称规则
- **唯一性**：客户名称必须唯一
- **长度限制**：2-20 个字符
- **字符限制**：支持英文、数字、下划线

#### 4.1.2 密码规则
- **长度限制**：5-20 个字符
- **复杂度要求**：建议包含字母和数字
- **必填验证**：创建账号时密码必填

#### 4.1.3 邮箱规则
- **格式验证**：必须符合邮箱格式
- **唯一性**：邮箱地址建议唯一（可选）

#### 4.1.4 手机号规则
- **格式验证**：必须符合手机号格式（11位数字，1开头）
- **唯一性**：手机号建议唯一（可选）

### 4.2 账号状态管理规则

#### 4.2.1 状态定义
- **正常(0)**：账号可正常使用
- **停用(1)**：账号被停用，无法登录

#### 4.2.2 状态转换规则
- **正常 → 停用**：可随时停用账号
- **停用 → 正常**：可重新启用账号
- **停用账号**：无法登录系统，但数据保留

### 4.3 权限分配规则

#### 4.3.1 角色分配
- **多角色支持**：一个账号可分配多个角色
- **权限合并**：多个角色的权限合并生效
- **角色验证**：分配的角色必须存在且启用

#### 4.3.2 部门关联
- **部门选择**：可选择所属部门
- **数据范围**：部门影响数据范围权限

### 4.4 密码重置规则

#### 4.4.1 重置权限
- **管理员权限**：仅管理员可重置密码
- **本人权限**：客户本人可在个人中心修改密码

#### 4.4.2 重置流程
- **重置方式**：管理员重置为临时密码
- **首次登录**：使用临时密码登录后需修改密码

### 4.5 账号删除规则

#### 4.5.1 删除条件
- **无关联数据**：账号无关联的业务数据
- **权限验证**：仅管理员可删除账号

#### 4.5.2 删除效果
- **软删除**：标记为已删除，数据保留
- **数据保留**：删除后数据保留一定时间（如30天）

## 5. 交互设计

### 5.1 页面布局

#### 5.1.1 搜索筛选区域
- **位置**：页面顶部
- **布局**：搜索表单 + 操作按钮
- **功能**：
  - 客户名称搜索
  - 手机号搜索
  - 状态筛选
  - 部门树筛选
  - 创建时间范围筛选

#### 5.1.2 数据表格区域
- **位置**：搜索筛选下方
- **布局**：表格 + 分页器
- **功能**：
  - 账号列表展示
  - 状态切换（开关）
  - 操作列（编辑、删除、重置密码）

### 5.2 账号创建/编辑对话框

#### 5.2.1 对话框入口
- **触发方式**：点击"新增"按钮或"编辑"按钮
- **打开方式**：弹窗（Dialog）
- **弹窗宽度**：800px

#### 5.2.2 表单布局
- **表单字段**：
  - 客户名称（必填）
  - 客户昵称（必填）
  - 部门选择（树形选择器）
  - 手机号码（格式验证）
  - 邮箱（格式验证）
  - 性别（单选）
  - 状态（单选）
  - 密码（创建时必填，编辑时可选）
  - 角色分配（多选）
  - 岗位分配（多选）
  - 备注

#### 5.2.3 表单验证
- **实时验证**：输入时实时验证格式
- **提交验证**：提交前验证必填字段
- **错误提示**：显示具体错误信息

### 5.3 密码重置对话框

#### 5.3.1 重置入口
- **触发方式**：点击表格中的"重置密码"按钮
- **打开方式**：确认对话框（MessageBox）

#### 5.3.2 重置确认
- **确认信息**：显示客户名称
- **确认按钮**：确认重置
- **取消按钮**：取消操作

#### 5.3.3 重置结果
- **成功提示**：显示临时密码（可选）
- **失败提示**：显示失败原因

## 6. 异常处理

### 6.1 数据加载异常

#### 6.1.1 网络错误
- **场景**：网络请求失败
- **处理**：显示错误提示，提供重试按钮

#### 6.1.2 数据为空
- **场景**：查询结果为空
- **处理**：显示空状态提示

### 6.2 账号操作异常

#### 6.2.1 创建失败
- **场景**：账号创建失败（如客户名重复）
- **处理**：
  - 显示错误提示
  - 保留表单数据
  - 允许重新提交

#### 6.2.2 删除失败
- **场景**：账号删除失败（如有关联数据）
- **处理**：
  - 显示错误提示
  - 说明失败原因

## 10. 接口定义

### 10.1 获取账号列表

**接口路径**：`POST /api/system/user/list`

**请求参数**：
```typescript
{
  pageNum: number;
  pageSize: number;
  userName?: string;
  phonenumber?: string;
  status?: string;
  deptId?: number;
  createTime?: string[];
}
```

**响应数据**：
```typescript
{
  code: number;
  msg: string;
  data: {
    list: UserInfo[];
    total: number;
  };
}
```

### 10.2 创建账号

**接口路径**：`POST /api/system/user/add`

**请求参数**：
```typescript
{
  userName: string;
  nickName: string;
  password: string;
  deptId?: number;
  phonenumber?: string;
  email?: string;
  sex?: string;
  status: string;
  roleIds: number[];
  postIds: number[];
  remark?: string;
}
```

### 10.3 更新账号

**接口路径**：`PUT /api/system/user/edit`

**请求参数**：
```typescript
{
  userId: number;
  nickName: string;
  deptId?: number;
  phonenumber?: string;
  email?: string;
  sex?: string;
  status: string;
  roleIds: number[];
  postIds: number[];
  remark?: string;
}
```

### 10.4 重置密码

**接口路径**：`PUT /api/system/user/resetPwd`

**请求参数**：
```typescript
{
  userId: number;
}
```
