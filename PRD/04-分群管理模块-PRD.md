## 1. 产品概述

### 1.1 模块定位

分群管理模块是 C360 客户全景平台的客户分群统一管理模块，支持分群创建、编辑、预览、推送商机、导出等功能。提供分群状态管理和审核流程，实现客户分群的统一管理和应用。

### 1.2 业务价值

- **分群统一管理**：集中管理所有客户分群，便于查找和使用
- **分群状态跟踪**：实时跟踪分群计算状态和生效状态
- **商机转化**：支持将分群推送为商机，实现营销转化
- **数据导出**：支持分群数据导出，用于营销活动

### 1.3 目标客户

- **营销人员**：管理客户分群、推送商机
- **运营人员**：查看分群状态、导出分群数据
- **数据分析师**：分析分群效果、优化分群策略

## 4. 业务规则

### 4.1 分群创建规则

#### 4.1.1 分群名称规则
- **唯一性**：分群名称必须唯一
- **长度限制**：分群名称长度 2-50 个字符
- **字符限制**：支持中文、英文、数字、下划线

#### 4.1.2 规则配置规则
- **必填验证**：必须配置分群规则
- **规则验证**：保存前验证规则有效性
- **规则来源**：可从"客户筛选与分群"模块保存而来

### 4.2 分群状态流转规则

#### 4.2.1 状态流转图
```
草稿(draft) → 计算中(running) → 生效(active)
                ↓
             失败(failed)
```

#### 4.2.2 状态转换规则
- **草稿 → 计算中**：保存分群后，如果设置了"保存后立即计算"，自动触发计算
- **计算中 → 生效**：计算成功，分群生效
- **计算中 → 失败**：计算失败，显示失败原因
- **失败 → 计算中**：可重新触发计算

### 4.3 分群计算规则

#### 4.3.1 计算触发规则
- **手动触发**：点击"计算"按钮触发
- **自动触发**：保存分群时，如果设置了"保存后立即计算"
- **定时触发**：支持定时任务自动计算（可选）

#### 4.3.2 计算过程规则
- **计算时间**：根据数据量，计算时间可能较长
- **进度显示**：计算过程中显示进度
- **结果通知**：计算完成后通知客户

### 4.4 推送商机规则

#### 4.4.1 推送条件
- **分群状态**：分群必须处于"生效"状态
- **成员数量**：分群成员数 > 0
- **权限验证**：客户必须有推送商机权限

#### 4.4.2 推送流程
- **确认提示**：推送前显示确认提示
- **商机创建**：为每个分群成员创建商机
- **推送结果**：显示推送成功/失败数量

### 4.5 导出规则

#### 4.5.1 导出条件
- **分群状态**：分群必须处于"生效"状态
- **成员数量**：分群成员数 > 0

#### 4.5.2 导出格式
- **导出格式**：Excel (.xlsx)
- **导出字段**：可自定义选择导出字段
- **数据脱敏**：导出数据根据权限进行脱敏处理

## 5. 交互设计

### 5.1 页面布局

#### 5.1.1 统计面板区域
- **位置**：页面顶部
- **布局**：5个统计卡片横向排列
- **卡片内容**：
  - 分群总数
  - 总成员数
  - 生效分群数
  - 草稿分群数
  - 已推送商机数

#### 5.1.2 搜索筛选区域
- **位置**：统计面板下方
- **布局**：搜索表单 + 操作按钮
- **功能**：
  - 分群名称搜索
  - 状态筛选
  - 创建人筛选

#### 5.1.3 数据表格区域
- **位置**：搜索筛选下方
- **布局**：表格 + 分页器
- **功能**：
  - 分群列表展示
  - 状态标签展示
  - 操作列（编辑、预览、推送商机、导出）

### 5.2 分群创建/编辑对话框

#### 5.2.1 对话框入口
- **触发方式**：点击"新建分群"按钮或"编辑"按钮
- **打开方式**：弹窗（Dialog）
- **弹窗宽度**：1000px

#### 5.2.2 表单布局
- **表单字段**：
  - 分群名称（必填）
  - 分群描述（可选）
  - 规则配置（必填，规则编辑器）
  - 保存后立即计算（复选框）
  - 计算完成后推送商机（复选框）

#### 5.2.3 规则编辑器
- **布局**：树状结构展示
- **操作**：
  - 添加条件
  - 添加分组
  - 删除节点
  - 拖拽排序
- **覆盖预估**：规则配置后可点击"覆盖预估"查看预估结果

### 5.3 分群预览

#### 5.3.1 预览入口
- **触发方式**：点击表格中的"预览"按钮
- **打开方式**：弹窗（Dialog）
- **弹窗宽度**：1200px

#### 5.3.2 预览内容
- **成员列表**：显示分群成员列表（分页）
- **成员信息**：OneID、姓名、性别、标签等
- **导出按钮**：支持导出当前预览的成员数据

### 5.4 推送商机对话框

#### 5.4.1 推送入口
- **触发方式**：点击表格中的"推送商机"按钮
- **打开方式**：确认对话框（MessageBox）

#### 5.4.2 推送确认
- **确认信息**：显示分群名称和成员数量
- **确认按钮**：确认推送
- **取消按钮**：取消操作

#### 5.4.3 推送结果
- **成功提示**：显示推送成功数量
- **失败提示**：显示推送失败数量和原因

## 6. 异常处理

### 6.1 数据加载异常

#### 6.1.1 网络错误
- **场景**：网络请求失败
- **处理**：显示错误提示，提供重试按钮

#### 6.1.2 数据为空
- **场景**：查询结果为空
- **处理**：显示空状态提示

### 6.2 分群计算异常

#### 6.2.1 计算失败
- **场景**：分群计算失败
- **处理**：
  - 显示失败原因
  - 分群状态更新为"失败"
  - 提供"重新计算"按钮

#### 6.2.2 计算超时
- **场景**：计算时间过长
- **处理**：
  - 显示计算进度
  - 允许取消计算
  - 支持后台计算

## 10. 接口定义

### 10.1 获取分群列表

**接口路径**：`POST /api/segment/list`

**请求参数**：
```typescript
{
  pageNum: number;
  pageSize: number;
  name?: string;
  status?: SegmentStatus;
  creator?: string;
}
```

**响应数据**：
```typescript
{
  code: number;
  msg: string;
  data: {
    list: SegmentInfo[];
    total: number;
  };
}
```

### 10.2 创建/更新分群

**接口路径**：`POST /api/segment/add` 或 `PUT /api/segment/edit`

**请求参数**：
```typescript
{
  id?: string;
  name: string;
  description?: string;
  ruleConfig: RuleConfig | null;
  triggerCalc?: boolean;
  triggerPush?: boolean;
}
```

### 10.3 分群预估

**接口路径**：`POST /api/segment/estimate`

**请求参数**：
```typescript
{
  ruleConfig: RuleConfig | RuleNode;
}
```

**响应数据**：
```typescript
{
  code: number;
  msg: string;
  data: {
    estimatedCount: number;
    estimatedRate: string;
    sampleData: Array<{
      oneId: string;
      name: string;
      matchReason: string;
    }>;
  };
}
```

### 10.4 推送商机

**接口路径**：`POST /api/segment/:id/push`

**请求参数**：
```typescript
{
  id: string;
}
```
