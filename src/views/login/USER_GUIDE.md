# CompanyX IDaaS 统一身份认证系统 - 客户使用手册

## 1. 系统简介
CompanyX IDaaS (Identity as a Service) 平台为您提供安全、高效、统一的身份认证服务。本系统采用了最新的**高级数据流 (Premium Data Stream)** 视觉设计，结合动态交互背景，在确保数据安全合规的前提下，为您带来现代化的数字体验。

## 2. 界面概览

登录页面主要由以下四个区域组成：

### 2.1 沉浸式驾驶舱背景
- **视觉体验**：采用了全屏高清背景 (`loginBg.png`)，叠加了**动态光流与粒子网络**动画。
- **交互效果**：背景中的光流线条与悬浮粒子会随着鼠标移动产生微妙的**视差与斥力响应**，寓意着数据的流动与连接。

### 2.2 顶部状态栏 (Header)
- **品牌标识**：左侧展示 CompanyX IDaaS 系统标识。
- **系统状态**：实时显示系统运行状态（如 `🟢 Operational`），点击可查看详细服务健康度。
- **国际化支持**：
    - 点击右上角 🌐 图标。
    - 支持 **简体中文 (Simplified Chinese)**、**繁体中文 (Traditional Chinese)** 和 **English** 实时切换。

### 2.3 登录核心区 (Login Card)
位于页面中央的白色卡片区域，支持两种认证方式：

#### 🔐 单点登录 (SSO Login)
*默认推荐方式*
- 适用于集团内部员工。
- 点击 **"SSO 登录"** 按钮（蓝紫渐变色），系统将自动跳转至企业统一授权中心。
- 认证成功后自动回调并登录系统，无需记忆额外密码。

#### 👤 本地账号登录 (Local Login)
*适用于管理员或特定合作伙伴*
- **账号输入**：支持客户名/邮箱/手机号。
- **密码安全**：具有密码显示/隐藏功能。
- **操作按钮**：
    - **登录**：高亮渐变色按钮，点击进行验证。
    - **重置**：清空当前输入内容。
- **服务协议**：登录前需勾选“同意服务协议与隐私政策”。

### 2.4 底部合规与版权区 (Footer)
- **地区/合规选择器**：
    - 提供 **CN (中国大陆)**、**HK (中国香港)**、**MO (中国澳门)** 三个选项。
    - **作用**：选择您所属的运营区域，系统将自动适配对应的**隐私法律条款**（如 PIPL 或 PDPO）及数据合规策略。
    - *样式*：选中项会以**白色胶囊卡片**高亮显示，代码加粗，清晰易读。
- **版权信息**：显示当前版本及版权所有年份。
- **快捷链接**：帮助文档 | 隐私条款 | 技术支持。

## 3. 操作指引

### 如果您是首次登录
1. **确认区域**：在页面底部确认当前选择的区域（CN/HK/MO）与您所在的办公地点一致。
2. **阅读协议**：建议点击“隐私保护指引”了解我们如何保护您的数据。
3. **选择方式**：
   - 集团员工请直接使用 **SSO 模式**。
   - 外部人员请切换至 **本地账号** 标签页输入授权账号。

### 常见问题 (FAQ)

**Q: 为什么背景会有线条在动？**
A: 这是系统内置的“高级数据流”可视化效果，象征着实时的数据传输与安全监控。它经过性能优化，不会影响您的电脑运行速度。

**Q: 为什么需要选择 CN/HK/MO 地区？**
A: 不同地区的数据隐私法律（如中国大陆的《个人信息保护法》）要求不同。为了保障您的数据权益及合规性，请务必选择正确的所属区域。

**Q: 无法点击登录按钮？**
A: 请检查输入框下方的“我已阅读并同意”勾选框是否已选中。
